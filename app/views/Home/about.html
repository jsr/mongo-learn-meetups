Find cool events, Learn MongoDB

Hacker Meetups is built by 10gen as a sample application to help people get familiar with MongoDB. We also hope it'll help people find and spread the word about upcoming tech-events - from large conferences to small user group get-togethers.

This is the first of, what we hope, will be many such application. Our goal is to build engaging learning resources which are also useful and neat. You can read about our philosophy on sample apps on our blog.

Hacker Meetups is written using the Play framework and, of course, MongoDB. The source code is available on GitHub.
